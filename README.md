# Finger Typing Tutor

Браузерный тренажёр слепой печати для обучения быстрой и точной печати без взгляда на клавиатуру.

## Возможности

- **Две раскладки**: английская (QWERTY) и русская (ЙЦУКЕН)
- **Четыре режима тренировок**:
  - **Select Keys** — тренировка на выбранных пользователем клавишах
  - **Random Key** — случайные символы из полного набора клавиш
  - **Phrases** — тренировка на фразах из словаря
  - **Files** — тренировка на тексте из загруженных файлов с сохранением прогресса
- **Сохранение прогресса**: прогресс по файлам сохраняется в localStorage
- **Визуальная обратная связь**: подсветка ошибок и отображение введённого текста

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен-сборки
npm run preview
```

## Режимы тренировок

### Select Keys
Выберите клавиши, на которых хотите тренироваться. Приложение будет генерировать случайные последовательности из выбранных символов.

### Random Key
Тренировка на случайных символах из полного набора допустимых клавиш текущей раскладки.

### Phrases
Тренировка на фразах, состоящих из двух случайных слов и знака препинания. Используются встроенные словари для английского и русского языков.

### Files
Загрузите текстовый файл и тренируйтесь на его содержимом. Прогресс сохраняется автоматически, вы можете вернуться к файлу позже и продолжить с того места, где остановились.

## Особенности

- Автоматическая загрузка следующей цели при правильном вводе
- Поддержка базовых клавиш для левой и правой руки
- Фильтрация недопустимых символов при работе с файлами
- Адаптивный интерфейс для длинных строк
- Возврат в режим настройки по клику в любом месте страницы
